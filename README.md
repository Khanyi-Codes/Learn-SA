# 🇿🇦 Learn SA

> Interactive curriculum-aligned learning for South African students — built on the CAPS framework.

---

## 📖 About

**Learn SA** is a free, web-based learning platform designed for South African learners from Grade R through Grade 12. It delivers interactive lessons, video content, and quizzes aligned to the **Curriculum and Assessment Policy Statement (CAPS)** — the official South African school curriculum.

Whether you're catching up, revising, or getting ahead, Learn SA meets you where you are.

> ⚠️ **Status: Work in Progress** — This project is actively being built. Features and content are being added progressively.

---

## ✨ Features (Planned & In Progress)

- 📚 **CAPS-Aligned Lessons** — Interactive lessons mapped to the official South African curriculum for all grades
- 🎥 **Video Lessons** — Engaging video content to support different learning styles
- 🧠 **Quizzes** — Grade and subject-specific quizzes to test and reinforce understanding
- 🏫 **All Grades Covered** — Content for Grade R through Grade 12
- 📱 **Responsive Design** — Works on desktop, tablet, and mobile devices
- 🆓 **Free to Use** — Built with accessibility and equity in mind

## 🚀 Getting Started

### Prerequisites
All you need is a modern web browser. No installations required.

### Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Khanyi-Codes/learn-sa.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd learn-sa
   ```

3. **Open the app**
   ```bash
   open index.html
   ```
   Or simply drag `index.html` into your browser.
